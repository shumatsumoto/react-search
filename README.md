## :pencil2: Overview

You can search for an account with the api on github.

## :hammer: Requirement

- macOS
- React 17.0.1
- [Material UI](https://material-ui.com/)

## :pushpin: Usage

Git clone
```
$ git clone https://github.com/shumatsumoto/react-search.git
```
Move to directory
```
$ cd react-search
```
Install the library
```
$ npm install
```
Launch a local server
```
$ npm run start
```
Access URL： 
http://localhost:3000
