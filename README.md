## :pencil2: Overview

I'm using Rails 6 rich text.
I created a simple SNS application.

## :hammer: Requirement

- macOS
- Rails 6.0.3.5
- [Semantic UI](https://semantic-ui.com/)

## :pushpin: Usage

Git clone
```
$ git clone https://github.com/shumatsumoto/rails-blog.git
```
Move to directory
```
$ cd rails-blog
```
Install the library
```
$ bundle install
```
Launch a local server
```
$ rails s
```
Access URL： 
http://localhost:3000

## :railway_car: Features

- Article posting
- Likes
- Rich text
- Login / Logout
- SignUp
